# Array Dinamis manual
Tujuannya adalah membuat array dinamis menggunakan array statis di bahasa pemrograman  
1. C
2. C++
3. Rust
4. .NET (VB dan C#)

Manfaatnya adalah, mengetahui, bahwa dibalik array dinamis, sebenarnya ada array statis yang bekerja dibaliknya.  
menggunakan looping dan if-else.  

Peringatan :  
Kode di atas hanya untuk pembelajaran, dan rasa penasaran bagaimana cara array dinamis bekerja di tingkat yang lebih rendah. Untuk standar industri, bisa memakai vector dan struct.
